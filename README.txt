# 種田小遊戲

這個遊戲模擬了一個簡單的農場，玩家可以耕種、施肥、收割果實，並通過商店功能購買種子和肥料。遊戲中，玩家需要管理金錢和資源，以擴展農場並收穫更多果實。

## 功能

1. **商店**：
   - 初始金錢：100 元，種子和肥料各 5 個。
   - 每次交易可透過勾選 CheckBox 選擇買入或賣出商品，按下按鈕完成交易。
   - 賣出果實可獲得 40 元，買種子或肥料需花費 10 元。
   - 若金錢不足，按鈕不會生效。
   
2. **農場**：
   - 程式啟動時生成 3x4 的農場 (Button List)，每個按鈕表示一塊田地，有四種狀態：泥土、種子、作物、果實。
   - 使用工具操作田地，包括澆水、撒種、施肥及收割。
     - **撒種**：將田地狀態由泥土變為種子。
     - **澆水+施肥**：將種子狀態轉為作物，再次澆水+施肥則會轉為果實。
     - **收割**：用鐮刀收割果實，果實數量 +1，並將田地狀態重置為泥土。

## 工具使用說明

- 工具包括澆水壺、種子、肥料和鐮刀，使用後相應物品數量會減少。
- 若種子或肥料用完，將顯示提示訊息「種子/肥料用完了」。
- 選取一種工具後，按下田地即可使用該工具對田地進行操作。
